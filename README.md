# Python + C++ Path Tracer (Nanobind)

A high-performance path tracer combining Python's ease of use with a C++ core engine.

## Features
*   **Hybrid Architecture**: Scene definition and orchestration in Python, compute-heavy rendering in C++ (via Nanobind).
*   **Advanced Rendering**:
    *   **Materials**: Glass (with Tint/IOR), Plastic (Clearcoat), Metal (Roughness), Lambertian, Diffuse Light.
    *   **Caustics**: "Fake" transparent shadows for realistic glass rendering.
    *   **Focus**: Depth of Field with adjustable aperture and focus distance.
*   **Preview Mode**: Real-time interactive viewer with camera controls (WASD + Mouse) to frame your shot.
*   **Denoising**: Integrated Intel Open Image Denoise (OIDN).
*   **Animation**: Turntable animation generation.

## Installation

This project uses `uv` for dependency management and orchestrating the build. The `setup_project.py` script automates the entire process, including downloading OIDN and compiling the C++ extension.

```bash
uv run setup_project.py
```

## Usage

All interaction is handled via the `main.py` entry point.

### 1. Interactive Preview
Launch the interactive viewer to fly around the scene and compose your shot.
```bash
uv run main.py --preview --scene random
```
**Controls**: `WASD` to move, `Mouse` to look.
**Output**: When you close the viewer window, the script prints the exact command used to render that specific view.

### 2. High-Quality Render
Run the renderer with your desired settings using the command generated by the preview or manually specified.
```bash
uv run main.py --scene random --width 1200 --height 800 --spp 100 --depth 50
```
*Outputs `outputs/images/`.*

### 3. Animation
Generate a turntable video around the scene center.
```bash
uv run main.py --scene random --animate --frames 48 --fps 24
```
*Outputs `outputs/videos/animation.mp4` and frames in `outputs/frames/`.*

## Command Line Arguments

*   `--scene`: Select scene (e.g., `cornell`, `random`, `showcase`).
*   `--preview`: Enable interactive viewer mode.
*   `--width`, `--height`: Output resolution.
*   `--spp`: Samples per pixel (higher = less noise).
*   `--depth`: Max bounce depth (higher = more accurate light transport).
*   `--auto-sun`: Enable procedural sun/sky lighting.
*   `--lookfrom`, `--lookat`, `--vup`: Camera positioning.
*   `--vfov`, `--aperture`, `--focus_dist`: Camera lens settings.

## Project Structure

*   `src/`: C++ Core source code (Renderer, BVH, Materials).
*   `modes/`: Python logic for `renderer` (offline) and `viewer` (interactive).
*   `scenes.py`: Python Scene definitions.
*   `main.py`: Main entry point.
*   `setup_project.py`: Build and setup automation script.
